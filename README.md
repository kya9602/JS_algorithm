## JS 알고리즘 문제 마라톤(스터디)

## 진행방식.
- 프로그래머스 난이도별 문제 공부 + 간단한 문제를 풀이 > 코드리뷰

## 느낀점.

- 스터디를 하면서 비 전공자 분들과 코드 리뷰를 하다보니 새로운 개념을 배우기도 했고 알고 있는 개념을 알려주면서 복습 할 수 있었다.( 알고리즘 실력의 편차에 상관없이 만족! )

- 공부 자료를 github repo에 정리했다.
